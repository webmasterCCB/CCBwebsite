# Classic City Band Website

## How to Upload This to GitHub

1. **Delete old files** in your GitHub repo (if any)
2. Go to: https://github.com/webmasterCCB/CCBwebsite
3. Click **"Add file"** → **"Upload files"**
4. **Drag ALL files and folders** from this ccb-website folder
5. Make sure you see folders like `_includes`, `_layouts`, `assets` in the upload
6. Click **"Commit changes"**
7. Wait 2-3 minutes
8. Your site will be live at: https://webmasterccb.github.io/CCBwebsite/

## Folder Structure

- `_includes/` - Header and footer
- `_layouts/` - Page templates
- `assets/css/` - Stylesheet
- `assets/images/` - Logo and images
- Root files - All your pages (.md and .html)
